# Project Summary

Ariadne is a lightweight AI coding framework and method pack for helping coding agents complete tasks more reliably.

Current direction:

- Framework, not platform.
- Progressive workflow depth.
- Short entry rules plus on-demand skills.
- Task workspaces preserve decision-grade context for Level 2 and Level 3 work.
- Sub-agents return decision-grade artifacts, while the main agent keeps final decision authority.
- Pending memory captures future-useful information during work before consolidation.
- Evidence-based task completion.
- Optional repo-map, memory, hooks, and sub-agent delegation.
