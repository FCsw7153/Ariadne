# Glossary

## Ariadne

A lightweight AI coding method pack for making complex agent tasks more reliable without becoming a heavy platform.

## Task Workspace

A persistent folder under `.ai/tasks/` that stores decision-grade context, plans, evidence, sub-agent outputs, and closure state for Level 2/3 tasks.

## Pending Memory

A real-time candidate pool for future reusable knowledge before consolidation.

## Decision-Grade Artifact

A concise artifact that helps a future agent or main agent make a safe decision. It includes facts, rationale, evidence, risks, and needed decisions.

## Main Agent

The orchestrator responsible for user alignment, scope, decisions, sub-agent acceptance, and final reporting.

## Sub-Agent

A context-isolated worker that performs focused research, implementation, review, verification, documentation, or memory capture.

## Evidence

Tests, commands, manual checks, review findings, changed-files summaries, artifact paths, or residual-risk notes used to support a completion claim.
